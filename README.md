# Countries

## Instructions

- Clone the repo
- Run **`npm install`**
- Run **`npm start`**

## Technologies

- React
- TypeScript
- Apollo GraphQL
- Ant Design

## Functions in the application

- Sorting of countries by their **`Name`**, **`ISO Code`** and **`Continent`**
- Search by **`Country Names`** and **`ISO Codes`**
- Filtering **`Countries`** by their **`Continents`**
